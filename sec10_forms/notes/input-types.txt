Forms input element types
  - <input> and it's attributes
    - Used for fetching users input.
        specifies what type of input will be used e.g. plain text, password, or email.
        - type=""
          - text
          - email
          - number
            - step=""
              - Determines incremented/decremented number
          - password
        - name=""
            - name="" attribute allows us to provide a unique identifier to an element.
            - name="" is important for form submission.
        - id=""
            - The id="" attribute connects an input to a label element.
        - value=""
          - value="" attribute allows us to fill in a value for an input.

  - <textarea>
    - Fetches multiline text input 
    - Used usually for comments, and feedback fields.
    - cols="30" rows="10"


  - Dropdown element <select>  and <option>
    <select>
      <option></option>
      <option></option>
      <option></option>
    </select>
   - A drop down multiple-choice/select inputs
   - name="" attribute is only added to the select element

  - <radio> type="radio"
    - Multiple choice options, where users get to select only one of these options.
    - name="" attribute for all radio button should be the exact same, in order to 
      ensure only one choice is selected.


  - <checkbox>
   - A checkbox button
   - Can be used to present a "toggle" (yes/no) option to the visitor (e.g. "Agree to terms")
